// Ana fonksiyon: Bir grup insan ve bir pasta parçası alır.
FONKSİYON AdilPaylasim(kisilerListesi, pasta):

    // 1. Fethet Adımı (Temel Durum):
    // Eğer listede sadece bir kişi varsa, pastanın tamamını o alır.
    EĞER kisilerListesi'nin eleman sayısı == 1 ISE:
        tekKalanKisi = kisilerListesi[0]
        tekKalanKisi.payi = pasta
        DÖN // Fonksiyonu bitir.
    
    // 2. Böl Adımı (İnsanları ve Pastayı Bölme):
    
    // Kişi listesini iki alt gruba ayır.
    N = kisilerListesi'nin eleman sayısı
    N1 = TAVAN(N / 2) // İlk grubun eleman sayısı
    
    grup1 = kisilerListesi'nin ilk N1 kişisi
    grup2 = kisilerListesi'nin geri kalan N - N1 kişisi

    // Şimdi pastayı bu iki grup arasında adil bir şekilde böl.
    // Bunun için "bir grup keser, diğeri seçer" yöntemi kullanılır.
    // Bu ara fonksiyon, grup1 ve grup2'nin temsilcileri arasında adil bir bölme yapar.
    (pastaParcasi1, pastaParcasi2) = GruplarArasiAdilBolme(grup1, grup2, pasta)
    
    // 3. Fethet Adımı (Özyineli Çağrılar):
    
    // Grup 1, kendisine düşen pastaParcasi1'i kendi içinde paylaşır.
    AdilPaylasim(grup1, pastaParcasi1)
    
    // Grup 2, kendisine düşen pastaParcasi2'yi kendi içinde paylaşır.
    AdilPaylasim(grup2, pastaParcasi2)

SON FONKSİYON


// Yardımcı fonksiyon: Pastayı iki grup arasında orantılı böler.
FONKSİYON GruplarArasiAdilBolme(grup1, grup2, tumPasta):
    // Grup1'in bir temsilcisi (örn: ilk üyesi) pastayı keser.
    // Kendi değerlendirmesine göre pastayı P1 ve P2 olarak ikiye ayırır.
    // P1'in değeri, grup1'in alması gereken oran olan (grup1.boyut / toplamKisiSayisi) kadardır.
    temsilci1 = grup1[0]
    (P1, P2) = temsilci1.KendiDegerineGoreKes(tumPasta, grup1.boyut / (grup1.boyut + grup2.boyut))

    // Grup2'nin temsilcisi bu iki parçadan daha değerli bulduğunu seçer.
    temsilci2 = grup2[0]
    EĞER temsilci2.Degerlendir(P1) > temsilci2.Degerlendir(P2) ISE:
        // Eğer grup2, grup1 için kesilen parçayı daha değerli bulursa,
        // grupların payları değişir. Bu, adil bir seçim olmasını sağlar.
        grup1inPayi = P2
        grup2ninPayi = P1
    ELSE:
        grup1inPayi = P1
        grup2ninPayi = P2
    
    DÖNDÜR (grup1inPayi, grup2ninPayi)
    
SON FONKSİYON

Kullanıcı, N kişi için adil pasta paylaşımı probleminin "Böl ve Fethet" yaklaşımı kullanılarak çözülmesini ve bu çözümün sözde kodunun yazılmasını talep etti. 
Yanıt olarak, N kişilik grubun özyineli bir şekilde iki alt gruba ayrıldığı, pastanın öncelikle bu iki grup arasında adil olarak bölündüğü ve ardından her grubun 
kendi payını yine aynı algoritmayla kendi içinde paylaştığı bir yöntem açıklandı. Bu teorik çözüm, algoritmanın adımlarını gösteren detaylı bir sözde kod ve 4 
kişilik bir senaryo üzerinden çalışan somut bir örnekle desteklenerek sunuldu.
